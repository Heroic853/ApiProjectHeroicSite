Heroic853 Site — Backend API

Tech Stack:

-ASP.NET Core Web API (C#)

-Entity Framework Core — gestione persistenza dati

-PostgreSQL — database principale

-Auth0 — autenticazione e gestione utenti (login, registrazione, cancellazione account)

-Stripe — pagamenti per commissioni e supporto

-Docker — containerizzazione e deploy

-JWT — validazione token per endpoint protetti

-CORS — comunicazione cross-origin con il frontend Blazor


Funzionalità principali:

-Autenticazione completa tramite Auth0: login, registrazione, cancellazione account

-Gestione ruoli — amministratore / utente con navigazione condizionale

-Pagamenti Stripe — commissioni Monster Hunter e donazioni di supporto

-API REST con endpoint protetti tramite JWT

-Gestione dati tramite Entity Framework Core su database PostgreSQL

-Containerizzazione con Docker per deploy su Railway e successivamente Render
